## How to run locally
1. Install dependencies:
    pip install -r requirements.txt
2. Create a .env file with:
    FLASK_ENV=production
    PORT=5000
3. Run the app:
    python app.py
4. Open:
    http://127.0.0.1:5000

## Deploy to Render.com
- Connect GitHub repo
- Set Build Command: pip install -r requirements.txt
- Set Start Command: python app.py

Enjoy!
